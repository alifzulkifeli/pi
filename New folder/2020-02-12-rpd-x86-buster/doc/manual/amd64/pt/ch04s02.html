4.2. Download dos Ficheiros a partir dos Mirrors Debian

Para encontrar o mirror mais próximo (e provavelmente o mais rápido), visite a lista de mirrors Debian.

4.2.1. Onde Encontrar Imagens de Instalação

As imagens de instalação estão localizadas em cada mirror Debian no directório debian/dists/buster/main/installer-amd64/current/images/ — o MANIFEST lista cada imagem e o seu propósito.
